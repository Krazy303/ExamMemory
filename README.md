# ExamMemory

Study and memory helper workspace powered by Cognee RAG.

## Features
- **Cognitive Subjects:** Create, search, and delete subjects dynamically.
- **AI-Powered Summarizations:** Summarizes answers into short, direct responses.
- **Enhanced Markdown:** Custom compilers for headings, emojis, task lists, and footnote formatting.

## Development

```bash
bun install
bun run dev
```

## Production Build

```bash
bun run build
```
